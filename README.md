# 苍穹外卖（RuBan）

个人独立手敲的完整外卖系统（毕设）

## 项目介绍
- 基于 SpringBoot3 + MyBatis-Plus + Redis + WebSocket + Docker
- 包含管理端 + 用户端 + 骑手端 + 微信小程序
- 完整业务：分类、套餐、菜品、订单、支付、地址簿、购物车、骑手接单、秒杀红包

## 技术栈
SpringBoot3 / MyBatis-Plus / Redis / MySQL8 / Nginx / Docker / WebSocket / 微信支付沙箱
